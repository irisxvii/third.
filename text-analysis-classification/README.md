## 1. Resume Info Extractor
- Built a Python script to extract emails and phone numbers from resumes using regular expressions.
- Tested on 3 sample resumes.
- Saved extracted data into output.json.

## 2. Sentiment Classification
- Loaded a custom sentiment CSV dataset.
- Preprocessed text: lowercasing, tokenization, stopword removal.
- Converted text into numerical form using TF-IDF.
- Trained Naive Bayes, SVM, and Random Forest classifiers.
- Evaluated models using Accuracy, Precision, Recall, and F1-Score.
- Visualized results using bar plots for model comparison.
